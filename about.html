About This Website

This project is an AI-powered system that analyzes radiology reports and automatically extracts medical image labels using Natural Language Processing (NLP). The system helps convert unstructured medical text into structured labels that can be used for medical image datasets and healthcare analytics.

Key Features

  • Automatic extraction of medical findings
  • NLP based radiology report analysis
  • Fast dataset labeling for medical images
  • Web-based interface using Flask

Technologies Used

  • Python
  • Flask
  • HTML & CSS
  • Natural Language Processing
